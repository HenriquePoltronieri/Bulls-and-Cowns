# 🎮 Bulls and Cows (Mastermind) - Kotlin

Projeto desenvolvido em **Kotlin** que implementa o jogo de adivinhação **Bulls and Cows** (similar ao Mastermind) no console.

## 📖 Descrição

O computador gera uma **palavra secreta** utilizando letras de um alfabeto pré-definido.  
O jogador deve tentar descobrir essa palavra dentro de um **número limitado de tentativas**.

Após cada tentativa, o jogo informa:

- ✅ **Acertos exatos** → letra correta na posição correta  
- 🔄 **Acertos parciais** → letra correta na posição errada

O jogo termina quando:

- 🏆 o jogador acerta a palavra secreta, ou  
- ❌ o número máximo de tentativas é atingido.

## 🛠 Tecnologias utilizadas

- Kotlin
- Kotlin/JVM
- IntelliJ IDEA
